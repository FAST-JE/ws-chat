## Project setup
```
npm i
```

### Start developers version
```
npm run dev
```

### Start node demon
```
npm run prod
```

### To view the logs
```
pm2 logs
```
### To stop
```
pm2 stop index
```
### For reload
```
pm2 reload index
```
### Response error codes
```
1001 - error in sql query
1002 - dialog not found
```
